## Run Locally

1- Clone the project

```bash
  git clone https://github.com/malekelkssas/Marvel-Ultimate-War-Game.git
```

2- Go to the project directory


3- Build the project using Maven
```bash
  mvn clean
```

4- Run Javafx project using Maven
```bash
  mvn javafx:run
```
